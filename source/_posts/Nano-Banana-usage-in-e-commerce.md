## Nano Banana 在电商中的使用

### 介绍

* ```Nano Banana```其正式名称为```Gemini 2.5 Flash Image```，是```Google```开发的一款```AI```图像生成和编辑工具，它整合了先进的图像互动生成与编辑能力，可以通过自然语言进行对话式编辑，并支持多图合成与推理，其中最风靡全球的是它的```3D```手办生成能力，由于其可高度保持图像一致性以及编辑能力，因此可应用于电商场景中，以提升商品的运营效率和营销效果。

### 案例

* 商品主体换背景，告别单调的白底图，将您的产品无缝融入到各种生活化或创意场景中，极大提升商品主图的吸引力和点击率：

  ```bash
  Replace the background in Image 1 with Image 2 and output a professional e-commerce product photo. Keep the product in Image 1 unchanged.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/商品主体换背景.png)

* 模特试装，仅需一张模特图和多件服装图，即可批量生成该模特试穿不同服装的效果图。这极大地节省了雇佣模特、租赁场地和专业拍摄的巨额费用：

  ```bash
  The model in image 1 wears the clothes in image 2 and strikes a professional model pose to show the design of the clothes.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/模特试装.png)

* 饰品配件替换，在不改变模特、背景和主体的情况下，快速替换图片中的特定元素，如项链、包包、眼镜、手机壳等。这对于展示产品的不同颜色、款式或搭配效果极为高效。

  ```bash
  Replace the necklace in image 1 with the necklace in image 2 and show the model in image 1 wearing the new necklace. Please replace it according to the style of the project in image 2. Do not modify it at will.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/饰品配件替换.png)

* 展示商品，让模特以某种方式展示您的产品，生成高度一致且真实的场景展示图。这对于需要展示使用方式或尺寸对比的产品尤为重要。

  ```bash
  The model in image 1 is carrying the bag in image 2, placing it naturally in front of her, and striking a professional model pose.
  ```
  
  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/展示商品.png)

* 广告分镜，将一张核心产品图延展为多格漫画或故事板，用于社交媒体广告或平台详情页，以更丰富、更动态的形式展示产品在不同场景下的应用。

  ```bash
  Create four scene images for the model in the image, namely office scene, banquet scene, commercial street scene, and home scene. Please keep the model's appearance and dress unchanged.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/广告分镜1.png)

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/广告分镜2.png) 

* 高清商品图，将分辨率较低的图片（如1688、淘宝的供应商图）进行高清化处理，智能提升图片质感、优化光影、增强纹理细节，使其达到高端跨境电商或品牌官网的主图标准。

  ```bash
  Improve the clarity, lighting, and textures of this image to make it look like a professional DSLR product shot.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/高清图片.png)

* 修复老素材，修复有划痕、噪点或褪色的老旧照片，并进行上色和高清化处理。这能帮助您重新利用积压的旧素材，节省重拍成本，或用于品牌怀旧营销。

  ```bash
  This vintage photo was restored by removing scratches, colorizing it, and upscaling it to 4K resolution.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/修复老素材.png)

* 虚拟试妆，对于美妆品牌，无需真人模特反复上妆卸妆，即可在一张模特素颜或淡妆图上，生成口红、眼影、腮红等产品的多种色号和妆容效果，直观展示产品卖点。这不仅极大地提高了内容生产效率，还能为不同肤色的模特生成效果图，满足全球市场的多元化需求。

  ```bash
  The model's face is adorned with a Japanese makeup look inspired by Satomi Ishihara: a dewy, glass-like base with a healthy glow. A soft peach-pink hue, blended with blush, is applied high on the apples of the cheeks. Eyes are finished with a soft brown satin eyeshadow, plumping and dewy the lower lids to create a lovely "silkworm" effect. A thin, natural brown inner liner is applied. Lips are plump, glossy, and hydrated, with a blurred coral-beige gradient. The overall look is fresh, innocent, and radiant. Soft, natural lighting is used.Please do not change the model's pose.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/虚拟试妆.png)

* 商品虚拟棚拍，只需要一张产品原图，就能为其生成拥有专业灯光、背景和创意的影棚级宣传照，彻底摆脱对摄影师和实体影棚的依赖。

  ```bash
  Create a high-end commercial image for this watch. Use dramatic chiaroscuro lighting to highlight the metallic finish. The background should be a dark, textured concrete wall. Add subtle water splashes around the watch.
  ```

  ![](https://raw.githubusercontent.com/Garden12138/picbed-cloud/main/ai/商品虚拟棚拍.png)
